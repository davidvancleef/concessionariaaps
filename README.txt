Projeto de faculdade da matéria de Algoritmos e Estruturas de Dados 2

O programa não utiliza funções de sistemas operacionais portanto em teoria deve funcionar em qualquer sistema.
O projeto está numa forma bem fácil de execução, a maioria dos comandos são dados por numeros de 1 a 5, geralmente nas partes de escolha o 1 será um CONFIRMA e o 0 será um CORRIGIR ou CANCELAR algo.
Há duas formas de uso, a entrada como gerente que garante acesso a funções exclusivas e mais dados dos funcionários e a entrada de vendedor que garante acesso apenas as funções de cadastrar venda, verificar as vendas feitas e ver os automóveis disponíveis para venda.
Após gerar a venda, o carro será removido da lista de automóveis e será gerada a nota fiscal (caso deseje).
Várias funcoes pedem a posiçao do funcionario/automovel. Para estas, caso nao se lembre do seu codigo de funcionario, deve-se utilizar o "0" para gerar uma lista apresentando a posição de cada um. Assim, você poderá responder a posição desejada, exemplo: 2

College project for Algorithms and Data Structures 2

The program does not use operating system functions, so in theory it should work on any system.
The project is very easy to execute, most of the commands are given by numbers from 1 to 5, where generally 1 will be a CONFIRM and 0 will be a CORRECT or CANCEL something.
There are two ways of using it, entering as a manager which guarantees access to exclusive functions and more employee data and entering as a salesperson which guarantees access only to the functions of registering sales, checking sales made and seeing cars available for sale.
After generating the sale, the car will be removed from the car list and an invoice will be generated (if desired).
Several functions ask for the operating/car position. For these, if you don't remember their operating code, you must use "0" to generate a list showing the position of each one. This way, you can respond to the desired position, example: 2
With the economic crisis in Brazil, the "Fire employee" function was added for the manager.
