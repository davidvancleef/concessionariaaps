O programa não utiliza funções de sistemas operacionais portanto em teoria deve funcionar em qualquer sistema.
O projeto está numa forma bem fácil de execução, a maioria dos comandos são dados por numeros de 1 a 5, geralmente nas partes de escolha o 1 será um CONFIRMA e o 0 será um CORRIGIR ou CANCELAR algo.
Há duas formas de uso, a entrada como gerente que garante acesso a funções exclusivas e mais dados dos funcionários e a entrada de vendedor que garante acesso apenas as funções de cadastrar venda, verificar as vendas feitas e ver os automóveis disponíveis para venda.
Após gerar a venda, o carro será removido da lista de automóveis e será gerada a nota fiscal (caso deseje).
Várias funcoes pedem a posiçao do funcionario/automovel. Para estas, caso nao se lembre do seu codigo de funcionario, deve-se utilizar o "0" para gerar uma lista apresentando a posição de cada um. Assim, você poderá responder a posição desejada, exemplo: 2
Com a crise econômica no Brasil, foi adicionada a função "Demitir funcionário" para o gerente.
